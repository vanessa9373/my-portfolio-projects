# D197 — Version Control

**Institution:** Western Governors University (WGU)
**Program:** B.S. Computer Science
**Status:** Completed

## Overview

Fundamentals and advanced use of Git for version control — branching strategies, merging, rebasing, pull requests, collaborative workflows, and managing repositories on GitHub.

## Topics Covered

- Git fundamentals: init, add, commit, push, pull
- Branching and merging strategies (GitFlow, trunk-based)
- Rebasing and resolving merge conflicts
- Pull requests and code review workflows
- GitHub Actions basics
- Repository management and best practices

## Tools & Technologies

- Git
- GitHub
- GitHub Actions
- Command Line / Bash
