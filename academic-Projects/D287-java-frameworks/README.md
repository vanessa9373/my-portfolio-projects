# D287 — Java Frameworks

**Institution:** Western Governors University (WGU)
**Program:** B.S. Computer Science
**Status:** Completed

## Overview

Building full-stack Java web applications using Spring MVC, Hibernate ORM, and Thymeleaf — covering dependency injection, JPA, and MVC architecture patterns.

## Topics Covered

- Spring MVC architecture and request lifecycle
- Dependency injection and Inversion of Control (IoC)
- Hibernate ORM and JPA for database persistence
- Thymeleaf templating engine
- Form handling and validation
- Session management
