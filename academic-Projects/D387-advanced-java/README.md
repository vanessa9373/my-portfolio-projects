# D387 — Advanced Java

**Institution:** Western Governors University (WGU)
**Program:** B.S. Computer Science
**Status:** Completed

## Overview

Advanced Java programming covering multithreading, concurrency, design patterns, the collections framework, generics, and enterprise-level application development.

## Topics Covered

- Multithreading and concurrency (ExecutorService, synchronized, locks)
- Design patterns (Singleton, Factory, Observer, Strategy)
- Java Collections Framework and Generics
- Lambda expressions and Streams API
- Exception handling and debugging
- Internationalization (i18n) and localization

## Tools & Technologies

- Java (JDK 17+)
- IntelliJ IDEA
- Maven
- JUnit 5
- Git / GitHub
