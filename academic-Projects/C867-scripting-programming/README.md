**Institution:** Western Governors University (WGU)
**Program:** B.S. Computer Science

## Overview

Applied programming and scripting in C++ — covering algorithms, data structures, file I/O, pointers, memory management, and building console-based applications.

## Topics Covered

- C++ syntax, data types, and control flow
- Pointers, references, and dynamic memory management
- Object-oriented programming in C++
- Data structures (arrays, linked lists, vectors, maps)
- File I/O and string manipulation
- Algorithm design and complexity analysis
