# D682 — AI Optimization for Computer Scientists

**Institution:** Western Governors University (WGU)
**Program:** B.S. Computer Science
**Status:** In Progress

## Overview

Optimization strategies for AI and machine learning systems — covering gradient descent, hyperparameter tuning, search algorithms, and computational efficiency for improving model performance.

## Topics Covered

- Gradient descent and variants (SGD, Adam, RMSProp)
- Hyperparameter optimization techniques
- Search algorithms (A*, genetic algorithms, simulated annealing)
- Constraint satisfaction and linear programming
- Computational complexity and efficiency
